# Portfólio Pessoal – Kailane Lisley

Este projeto consiste no desenvolvimento de um **site de biografia e portfólio pessoal**, criado como parte da disciplina **Programação Web e Mobile**.

O objetivo é apresentar informações acadêmicas, interesses de pesquisa, habilidades técnicas e projetos desenvolvidos ao longo da graduação.

Além do propósito acadêmico, o site também será utilizado como **portfólio pessoal**, podendo receber atualizações futuras com novos projetos e experiências.

## 🌐 Acesse o site

🔗 https://SEU-LINK-VERCEL-AQUI

## 📚 Sobre o projeto

O site apresenta:

* Informações **acadêmicas e profissionais**
* Seção **Sobre Mim**
* **Habilidades técnicas**
* **Projetos desenvolvidos**
* **Biografia pessoal**
* **Formulário de contato**

O layout foi pensado para ser **simples, moderno e responsivo**, funcionando em diferentes tamanhos de tela.
